# Contrast audit — Legacy theme

The Procore **Legacy** theme is a visual stopgap for the NGX transition and is **not** held to WCAG AA/AAA the same way as NGX Light. Prefer NGX Light for new work where accessibility is a hard requirement.

- **Skill metadata:** see `SKILL.md` frontmatter (`wcag: non-compliant`).
- **NGX path:** use the `ngx-design-system` skill and NGX token set for compliant themes.

Document known high-risk pairings (e.g. brand orange on white) in design reviews; do not treat Legacy as the long-term accessible baseline.
